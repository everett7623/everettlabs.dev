## What it solves

Server benchmarking often requires running several disconnected tools and manually copying the results. NodeLoc Bench provides a single launcher and a shareable result file.

## Who it is for

VPS operators and community members who need a quick, reproducible way to test server performance and network connectivity.

## Why it exists

NodeLoc is a long-standing community benchmark ecosystem. This repository wraps those tools into one launcher while preserving upstream attribution.

## Key capabilities

- Performance, IP, connectivity, routing, and network diagnostic tests
- Single launcher for multiple community tools
- Shareable result file output
- Maintained fork and integration
